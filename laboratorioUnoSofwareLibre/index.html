Introducción a PHP

¿Que es PHP?

PHP (Hypertext Preprocessor) es un lenguaje de programación muy popular para crear páginas web dinámicas. A diferencia de HTML o CSS, que son lenguajes estáticos, PHP se ejecuta en el servidor y puede conectarse a bases de datos, manejar formularios y generar contenido en tiempo real. Si alguna vez has iniciado sesión en una página, publicado en un foro o comprado en una tienda online, lo más probable es que en algún punto se haya usado PHP detrás de escena.

Instalación y entorno

Para trabajar con PHP no basta con escribir el código, necesitamos un servidor que lo ejecute. Si solo abres un archivo .php en tu navegador, no funcionará, porque PHP se interpreta en el lado del servidor.

Paso a paso (Windows con XAMPP):
  1. Descarga XAMPP e instálalo.
  2. Abre el panel de control de XAMPP y activa Apache y MySQL.
  3. Entra a la carpeta htdocs (ubicada dentro de XAMPP).
  4. Crea un archivo llamado hola.php y escribe lo siguiente:
<?php
          echo "Hola Mundo desde PHP";
          ?>

Luego abre tu navegador y visita: http://localhost/hola.php. Si ves el mensaje en pantalla, ¡felicidades, ya tienes PHP funcionando!

PHP tiene una sintaxis muy parecida a otros lenguajes como C o JavaScript, pero con algunas particularidades. Lo más básico que debes conocer:

  • Variables: Siempre empiezan con $. Ejemplo: $nombre = "Daniel";.
  • Tipos de datos: texto (string), números enteros (int), decimales (float), verdadero/falso (boolean), arreglos, objetos y null.
  • Operadores:
    • Aritméticos: + - * / %
    • Comparación: == != > < >= <=
    • Lógicos: && || !
  • Estructuras de control: if, else, switch. Estruturas de bucles for, while, foreach.

Ejemplo de Estructura de Control if
<
?php
$edad = 20;

if ($edad >= 18) {
echo "Eres mayor de edad.";
}
?>

Ejemplo de Estructura de Control if-else

<?php
$hora = 10;
if ($hora < 12) {
echo "Buenos días";
} else {
echo "Buenas tardes";
} ?>

Ejemplo de Estructura de Control Switch
<?php
$dia = 3;
switch ($dia) {
case 1:
echo "Lunes";
break;
case 2:
echo "Martes";
break;
case 3:
echo "Miércoles";
break;
default:
echo "Día desconocido";
} ?>

Ejemplo de Estructura de Bucle While
<?php
$contador = 1;
while ($contador <= 5) {
echo "Número: " . $contador;
$contador++;
} ?>

Ejemplo de Estructura de Bucle Do While
<?php
$contador = 1;
do {
echo "Número: " . $contador;
$contador++;
} while ($contador <= 5); ?>

Ejemplo de Estructura de For
<?php
for ($contador = 1; $contador <= 5; $contador++) {
echo "Número: " . $contador;
} ?>

Ejemplo de Estructura de Bucle Foreach
<?php
$numeros = [1, 2, 3, 4, 5];
foreach ($numeros as $numero) {
echo "Número: " . $numero;
} ?>

Funciones en PHP

Una función es un bloque de código que puedes reutilizar. Esto evita repetir lo mismo una y otra vez.

Ejemplo de función:

<?php
function saludo($nombre) {
    return "Hola, " . $nombre . "!";
}
echo saludo("Cristian"); // Hola, Cristian!
?>

Consejo: las funciones te ayudan a organizar tu programa en partes más pequeñas y fáciles de entender.

Manejo de Formularios

PHP se usa mucho para procesar datos de formularios, como logins o registros.

Ejemplo:

<form method="post">
  Nombre: <input type="text" name="usuario">
  <input type="submit" value="Enviar">
</form>

<?php
if(isset($_POST['usuario'])){
    echo "Bienvenido, " . $_POST['usuario'];
}
?>

Nota: siempre valida y limpia los datos que recibes para evitar ataques de seguridad.

Conexión con Bases de Datos

Uno de los puntos más fuertes de PHP es que puede trabajar con bases de datos como MySQL. Aquí un ejemplo sencillo usando MySQLi: 

<?php
$conn = new mysqli("localhost", "root", "", "mi_base");

if ($conn->connect_error) {
  die("Error de conexión: " . $conn->connect_error);
}

$result = $conn->query("SELECT nombre FROM usuarios");

while($row = $result->fetch_assoc()) {
  echo $row["nombre"] . "<br>";
}
$conn->close();
?>

En este ejemplo: - Nos conectamos al servidor local. - Ejecutamos una consulta. - Recorremos los resultados con un bucle while. - Cerramos la conexión.

Sesiones en PHP

Las sesiones permiten guardar información mientras un usuario navega por la página. Por ejemplo, recordar quién ha iniciado sesión. 

<?php
session_start();
$_SESSION["usuario"] = "Daniel";
echo "Bienvenido, " . $_SESSION["usuario"];
?>

Manejo de Archivos

PHP también puede crear, leer y modificar archivos en el servidor. 

<?php
$file = fopen("datos.txt", "w"); // Abrir en modo escritura
fwrite($file, "Hola desde PHP"); // Escribir
fclose($file); // Cerrar archivo
?>

Este código genera un archivo datos.txt y escribe en él el texto "Hola desde PHP".
